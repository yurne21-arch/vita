# Iconos PWA

Coloca aquí los iconos referenciados por `manifest.json` e `index.html`:

- `Icon-192.png` (192×192)
- `Icon-512.png` (512×512)
- `Icon-maskable-192.png` (192×192, con zona segura)
- `Icon-maskable-512.png` (512×512, con zona segura)

Puedes generarlos con cualquier generador de iconos PWA a partir del logo de VITA
(fondo blanco, acento verde oliva #6B7A4F). Son lo único pendiente para que el ícono
se vea bien al "Añadir a pantalla de inicio" en el iPhone.
