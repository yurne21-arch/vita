# ADR 0006 — Gratis primero

**Contexto.** Prioridad: mejor producto con la menor inversión durante el MVP.

**Decisión.** Prohibido adoptar herramientas de pago si una alternativa gratuita logra ≥90% del resultado. Todo gasto fijo requiere aprobación explícita. Cada recomendación técnica incluye su costo (mensual, anual, alternativa gratuita, ventajas, desventajas, cuándo pagar).

**Consecuencias.** El MVP opera a ~$0; las inversiones se deciden con datos de uso real.
